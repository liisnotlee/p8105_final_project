Process:

  1. Ensure each subject has ONLY one height and weight on file. If a subject contains multiple height and weight (both in 2022 and 2024), take the mean.
  2. Note missing data:
ht_wt = read_csv("data/height_and_weight.csv")|> 
  janitor::clean_names() |> 
  rowwise() |>
  mutate(
    height = mean(c(height_2022, height_2024), na.rm = TRUE),
    weight = mean(c(weight_2022, weight_2024), na.rm = TRUE)
  ) |>
  ungroup() |> 
  select(id, height, weight)
